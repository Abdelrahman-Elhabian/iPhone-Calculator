# iPhone Calculator Simulator

A pixel-perfect, fully functional **iPhone-style calculator** built with **pure HTML, CSS, and JavaScript** — no frameworks, no libraries!


## Features

- Authentic iPhone calculator design & colors
- Smooth button animations on click
- Real-time operation display
- Supports: `+`, `−`, `×`, `÷`, `%`, `.` (decimal)
- Clear (C) button resets everything
- Beautiful number formatting with commas (e.g., 1,234,567)
- Error handling ("Error" on invalid expressions)
- Responsive & mobile-friendly
- Zero dependencies — just vanilla JS!
